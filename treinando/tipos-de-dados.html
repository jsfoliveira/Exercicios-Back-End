-- tipos de dados -- BOOLEANOS: tinyint(1) Se não passar nada, será null. Pode inserir 1 ou 0. -- CARACTERES CHAR(5): permite inserir até 5 caracteres. se só escrever 2, poderá fazer, mas ocupará o espaço reservado de 5. VARCHAR(10): permite inserir até 10 caracteres. se só escrever 2, poderá fazer e ocupará o espaço reservado de 2. -- NUMEROS TINYINT(unsigned): permite adicionar valores de 0 a 255 positivos TINYINT(signed): permite adicionar valores de -128 a +127 SMALLINT MEDIUMINT INT BIGINT DECIMAL(5,2): pode definir um número com 5 números, com duas casas decimais, exemplo: 199.99 FLOAT/REAL: uma casa decimal só, exemplo: 10.9 DOUBLE: duas casas decimais, exemplo: 49.90 -- TEMPORAIS DATETIME: YYYY-MM-DD HH:MM:SS TIMESTAMP: igual ao datetime só que tem fuso horário
