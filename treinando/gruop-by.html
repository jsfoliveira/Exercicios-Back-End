-- Exibindo e filtrando dados de forma agrupada com GROUP BY e HAVING -- COUNT não conta valores null -- GROUP BY agrupa todos os registros que possuem o mesmo valor da coluna, evitando duplicações na hora de retornar os dados. -- Com isso, o GROUP BY vai retornar menos valores da coluna, pois ele agrupará valores iguais, evitando células com valores duplicados. SELECT coluna(s), count(*) FROM tabela GROUP BY coluna(s); -- Esses dois casos, vai mostrar quantas vezes aparece 'PG', terão os mesmos resultados SELECT rating, COUNT(*) FROM sakila.film GROUP BY rating; SELECT COUNT(*) FROM sakila.film WHERE rating='PG'; -- conte quantas vezes aparece o nome SELECT first_name, COUNT(*) AS quantidade FROM sakila.actor GROUP BY first_name; -- exemplo: Juca 9 -- Média de duração de filmes agrupados por classificação indicativa SELECT rating, AVG(length) FROM sakila.film GROUP BY rating; -- Valor mínimo de substituição dos filmes agrupados por classificação indicativa SELECT rating, MIN(replacement_cost) FROM sakila.film GROUP BY rating; -- Valor máximo de substituição dos filmes agrupados por classificação indicativa SELECT rating, MAX(replacement_cost) FROM sakila.film GROUP BY rating; -- Custo total(soma) de substituição de filmes agrupados por classificação indicativa SELECT rating, SUM(replacement_cost) FROM sakila.film GROUP BY rating; -- Monte uma query que exiba a quantidade de clientes cadastrados na tabela sakila.customer que estão ativos e a quantidade que estão inativos. SELECT active, COUNT(*) FROM sakila.customer GROUP BY active; -- Monte uma query para a tabela sakila.customer que exiba a quantidade de clientes ativos e inativos por loja. -- Os resultados devem conter o ID da loja, o status dos clientes (ativos ou inativos) e a quantidade de clientes por status. SELECT active, COUNT(*), store_id FROM sakila.customer GROUP BY store_id, active; -- Monte uma query que exiba a média de duração de locação por classificação indicativa (rating) dos filmes cadastrados na tabela sakila.film. -- Os resultados devem ser agrupados pela classificação indicativa e ordenados da maior média para a menor. -- precisa dar um nome ao agrupamento ou chamar da forma que ta no order by: ORDER BY AVG(rental_duration) DESC.
